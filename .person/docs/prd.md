# PRD: 멀티유저 블로그 플랫폼

## 문제 정의

누구나 가입해서 글을 쓰고 소통할 수 있는 공개형 멀티유저 블로그 플랫폼을 만든다.

---

## 대상 사용자

| 역할 | 설명 |
|------|------|
| 일반 회원 | 가입 후 글 작성, 댓글 작성 가능. 자신의 글/댓글만 수정·삭제 가능. |
| 비회원 | 글 목록, 글 상세, 검색, 댓글 조회만 가능. |

---


## 핵심 요구사항

### P0 - 필수

1. **회원가입 / 로그인**
   - username + password + email로 자체 가입
   - 소셜 로그인 없음
   - 회원가입 시 username 중복 체크 (AJAX)

2. **게시글 CRUD**
   - 로그인한 사용자만 작성 가능
   - 본인 글만 수정·삭제 가능

3. **댓글 CRUD**
   - 1단계 댓글 (대댓글 없음)
   - 본인 댓글만 수정·삭제 가능

4. **권한 관리**
   - 본인 글/댓글에만 수정·삭제 버튼 노출
   - 관리자 역할 없음

5. **게시글 목록 페이징**
   - 한 페이지에 5개씩 표시

### P1 - 중요

6. **제목 검색**
   - 게시글 제목 기준으로 검색
   - 검색 결과도 페이징 적용

---

## 범위 밖 (명시적 제외)

- 소셜 로그인
- 관리자 기능
- 태그 / 카테고리
- 대댓글
- 좋아요
- 무한 스크롤

---

## 수용 기준 (Acceptance Criteria)

- [ ] 회원가입 후 로그인 가능
- [ ] 로그인한 사용자가 글 작성/수정/삭제 가능
- [ ] 본인 글/댓글에만 수정·삭제 버튼 노출
- [ ] 비로그인 사용자는 글/댓글 작성 불가
- [ ] 글 목록에서 5개씩 페이징 동작
- [ ] 제목으로 검색 시 결과 목록 표시
- [ ] 댓글 작성/수정/삭제 가능
